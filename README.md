# Projeto-de-Aprendizado-de-Maquina-de-Ponta-a-Ponta
O projeto visa construir um modelo de Machine Learning supervisionado para prever o valor médio de imóveis nos distritos da Califórnia, usando dados demográficos do censo. O objetivo é substituir estimativas manuais imprecisas por um sistema de regressão automatizado capaz de orientar e melhorar as decisões de novos investimentos da sua empresa.
